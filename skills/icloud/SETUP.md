The user is asking about iCloud Calendar or Mail, but iCloud integration is **not yet configured** for their account.

## How to set up iCloud

To use iCloud Calendar and Mail through Marcel, the user needs to provide two things:

1. **Apple ID** — the email address used to sign in to iCloud
2. **App-specific password** — a special password generated at [appleid.apple.com](https://appleid.apple.com)

### Steps for the user

1. Go to [appleid.apple.com](https://appleid.apple.com) and sign in
2. Navigate to **Sign-In and Security** > **App-Specific Passwords**
3. Click **Generate an App-Specific Password**
4. Give it a label like "Marcel"
5. Copy the generated password
6. Tell Marcel: "Set up iCloud with my Apple ID [email] and password [password]"

Marcel will store these credentials securely in the user's private data directory.

### What becomes available after setup

- **Calendar** — view upcoming events across all calendars
- **Mail** — search inbox by keyword (subject and body)

Tell the user these steps clearly and offer to help them through the process.
